# Alert_Cabin
A fully software, multimodal AI system that reasons about driver risk in real time — no new sensors, no hardware, just the camera and microphone already in your phone or car.

//Vision Detection
Features:
- Eye closure detection (EAR)
- Yawning detection (normalized mouth ratio)
- Real-time webcam monitoring
- Alarm system for drowsiness

Run:
pip install -r requirements.txt
python main.py
